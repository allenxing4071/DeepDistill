# DeepDistill 经验沉淀 Skill

> **核心原则：踩过的坑不踩第二次，好的实践要固化下来。**

## 触发词
产出新经验/新坑位/新最佳实践时，按经验类型路由到对应 Skill。本文件作为无明确归属经验的暂存区。

## 经验路由表

| 经验类型 | 优先写入 |
|---|---|
| ASR/Whisper 坑位/精度问题 | `.skills/内容处理/SKILL.md` |
| OCR 识别问题/多语言 | `.skills/内容处理/SKILL.md` |
| 文档提取/格式兼容 | `.skills/内容处理/SKILL.md` |
| LLM 调用优化/prompt 技巧 | `.skills/AI分析/SKILL.md` |
| 视频分析模型/兼容性/精度 | `.skills/视频分析/SKILL.md` |
| 部署/环境/Docker/依赖 | `.skills/部署运维/SKILL.md` |
| 项目结构/架构变更 | `.skills/项目入门/SKILL.md` |
| 无明确归属 | **本文件** |
